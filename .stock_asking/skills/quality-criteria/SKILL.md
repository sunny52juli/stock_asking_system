# 结果质量评估标准

## 候选数量评估

| 候选数量 | 评分 | 说明 |
|---------|------|------|
| 0 | 0.0 | 无候选，完全失败 |
| 1-5 | 0.4 | 过少，可能条件太严 |
| 6-9 | 0.6 | 偏少，可接受 |
| 10-30 | 1.0 | **最优区间** |
| 31-50 | 0.7 | 偏多，建议增加过滤 |
| >50 | 0.3 | 过多，条件太松 |

## 行业分散度评估

```
行业分散度 = 不同行业数量 / min(候选总数, 10)
```

- **>0.7**：优秀（1.0分）
- **0.5-0.7**：良好（0.7分）
- **<0.5**：较差（0.4分）

## 回测指标评估（如有）

- **Sharpe Ratio > 0**：正向收益风险比（+0.3分）
- **最大回撤 < 30%**：风险控制良好（+0.3分）
- **胜率 > 50%**：策略有效性强（+0.2分）

## 综合质量分数

```
quality_score = (
    候选数量评分 * 0.4 +
    行业分散度评分 * 0.2 +
    回测指标评分 * 0.2 +
    筛选条件合理性 * 0.2
)
```

**决策规则**：
- `quality_score >= 0.7`：通过，返回结果
- `0.5 <= quality_score < 0.7`：警告，但仍返回
- `quality_score < 0.5`：不通过，调整条件重试

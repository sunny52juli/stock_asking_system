# 股票筛选技能

## 核心原则

1. **明确筛选条件**：将用户的自然语言需求转化为具体的量化指标
2. **合理设置阈值**：避免过于宽松（候选太多）或过于严格（候选太少）
3. **考虑行业分散**：单一行业不超过候选总数的 50%
4. **避免未来函数**：只能使用筛选日期之前的数据

## 常用筛选维度

### 量价指标
- **成交量放大**：`vol / avg_volume_5d > 1.5`（当日成交量是5日均量的1.5倍）
- **价格突破**：`close > high_20d * 0.98`（接近20日最高价）
- **涨跌幅**：`pct_chg > 0.03`（单日涨幅超过3%）
- **换手率**：`turnover_rate > 0.05`（换手率超过5%）

### 技术指标
- **RSI**：`rsi_14 < 30`（超卖）或 `rsi_14 > 70`（超买）
- **MACD**：`macd > macd_signal`（金叉）
- **均线**：`close > ma_20`（站上20日均线）

### 基本面指标
- **市盈率**：`pe_ratio < 20`（低估值）
- **市净率**：`pb_ratio < 2`（低市净率）
- **ROE**：`roe > 0.15`（高净资产收益率）

## 筛选流程

1. **理解意图**：识别用户的核心需求（放量、突破、低估等）
2. **选择指标**：从上述维度选择合适的指标组合
3. **设置阈值**：根据经验设置合理的阈值范围
4. **执行筛选**：调用 `run_screening` 工具
5. **评估结果**：检查候选数量是否合理（理想区间 10-30 只）

## 示例

**用户查询**："找出近期放量上涨的股票"

**筛选逻辑**：
```json
{
  "strategy_name": "volume_breakout",
  "screening_logic": {
    "expression": "(pct_chg > 0.03) & (vol / avg_volume_5d > 1.5)",
    "confidence_formula": "pct_chg * 100 * 0.5 + (vol / avg_volume_5d - 1) * 100 * 0.5"
  }
}
```

**注意事项**：
- 确保使用的指标在可用字段列表中
- 表达式必须是有效的 Python 布尔表达式
- 置信度公式用于排序，值越大越靠前

# 数据质量规则

## 禁止未来函数

- 只能使用筛选日期**之前**的数据计算指标
- 不得使用未来的收盘价、成交量等信息
- 所有技术指标必须基于历史数据

## 停牌股票过滤

- 自动排除筛选日期停牌的股票
- 停牌判断依据：`vol == 0` 或 `suspended == True`

## 数据完整性检查

- 确保每只股票有至少 60 个交易日的历史数据
- 缺失数据超过 10% 的股票应被排除
- 异常值检测：价格突变超过 50% 需人工确认

## 数据时效性

- 优先使用最新可用的交易日数据
- 如果最新数据不可用，向前回溯最多 3 个交易日
- 超过 3 天无数据的股票视为无效

## 📊 核心指标范围表

| 工具名 | 取值范围 | 典型合理阈值 | 说明 |
| :--- | :--- | :--- | :--- |
| `outperform_rate` | `[0, 1]` | `> 0.5` | 跑赢比例 |
| `beta` | `(0, 2)` | `< 1.2` | 波动弹性 |
| `alpha` | `(-∞, +∞)` | `> -0.01` | 超额收益（小数） |
| `correlation` | `[-1, 1]` | `> 0.5` | 相关系数 |
| `tracking_error` | `[0, 0.5]` | `< 0.15` | 年化跟踪误差 |
| `information_ratio` | `[-2, 2]` | `> 0.5` | 信息比率 |
| `rsi` | `[0, 100]` | `30 - 70` | 相对强弱指标 |
| `volatility` | `[0, 1]` | `< 0.3` | 年化波动率 |
| `pct_change` | `(-1, +∞)` | `> 0.03` | 涨跌幅（小数） |

## ❌ 常见错误示例
- `rsi > 150` (越界)
- `outperform_rate > 1.5` (越界)
- `correlation > 1.2` (越界)
